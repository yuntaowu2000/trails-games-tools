## trails-games 
  
各类抓包，读取，批处理代码文件  

使用前先pip install -r requirements.txt

部分包含```sys.argv```的文件需使用命令行 python *.py 图片文件相对，或完整路径  
对于这些文件，可使用命令行批处理 ```for %f in (*.*) do python *.py %f```

[其他工具内容](https://github.com/trails-game)
